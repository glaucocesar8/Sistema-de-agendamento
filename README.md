# 💅 Sistema de Agendamento

Sistema web para gerenciamento de agendamentos de salão.

## 🚀 Tecnologias
- Node.js
- Express
- MySQL
- HTML, CSS e JavaScript

## 📌 Funcionalidades
- Agendamento de clientes
- Dashboard administrativo
- Login com JWT
- Cancelamento de agendamentos

## ▶️ Como rodar
npm install
node app.js
